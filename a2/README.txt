Buttons:
[1 through 5] 
    Changes the current buffer to use, where the number is the number of 
    vertices in the buffer 
[Random Colors] 
    Randomizes the uniform color passed to the fragment shader 
[+ & -] 
    Increase and decreases the speed of rotation 
[Stop] 
    stops the animation, setting the speed to 0
[Reset Angle] 
    resets the objects to their original angle
[gl.drawArrays Mode]
    Changes the rendering mode to the corresponding button

Other:
    Clicking and dragging causes the closest vertex in the rendered image to 
    change to the current mouse position. Changes the vertex in all of the
    buffers, not just the currently rendered one.
